Extra suggestions for learning this exam

  • Practice assessment first, then fill gaps. Take the free official Microsoft Learn practice assessment early to see which of the 5 domains needs the most work — then spend time proportional to each domain's weight (core infra and security are the two heaviest at 25–30% and 15–20%).
  • Lab it, don't just read it. Use the Microsoft Learn training path's free sandboxes to actually build a hub-and-spoke VNet, peer it, add a VPN Gateway and Firewall — routing/UDR questions are much easier once you've watched traffic actually take the wrong path once.
  • Use "Mark as reviewed" as a spaced-repetition tracker. Tick off a card only once you can explain its How column from memory; revisit unticked cards a few days later instead of re-reading everything linearly.
  • Summary mode = your final cheat sheet. Toggle it on the day before the exam and read top-to-bottom — it collapses every card to one line so you can sanity-check recall in under 15 minutes.
  • Print it. The page has a print stylesheet (Ctrl/Cmd+P) that expands everything and strips the UI chrome, so you can export a clean PDF cheat sheet.
  • Know your SKUs cold. A large share of scenario questions hinge on picking the right SKU/tier (VPN Gateway SKU, ExpressRoute tier, Load Balancer SKU, Application Gateway v2 vs WAF_v2, Front Door Standard vs Premium) — the How field on each card calls these out deliberately.
